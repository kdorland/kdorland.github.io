Small web experiments

  • Group Generator.
    I needed a small group generator at one point at work. I don't use it anymore, but here it is.
  • Java Source Printer.
    My attempt at making a web based Java code formatter with syntax highlighting. I needed it to generate HTML ready for printing without to much syntax coloring, and it had to highlight any Java code comment with my initials in it (using bright red). It works, but the print button is buggy. Keep hitting it a few times until it opens a tab that isn't empty (I need a better mechanism for this feature).

Fun and games

  • Mini Circle Wars - (2013)
    This was a Ludum Dare 26 entry. It's a tower defense game made in less than 48 hours using Java and libgdx.
  • Valhalla - Depths of Asgard - (2013)
    This is a silly little platforming game I made for the browser. I clearly have more talent when it comes to coding than doing graphics or designing levels. But I had a lot of fun creating the actual engine, though I never got to use it again after this game.
  • Atoms - (2000)
    A board game I made back in high school that was inspired by an old public domain board game I used to play on the Atari ST. It was programmed in C using libsdl, and I believe I made the graphics in Paint Shop Pro 4. What's amazing about this is that I made it for Windows 98, and it still works flawlessly on Windows 10.
