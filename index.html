Web experiments

  • Group Generator. Generate random groups of stuff from a custom list.
  • Java Source Printer. A web based Java code formatter with syntax highlighting using "highlight.js".
  • Value Noise Generator. An old tool I made to generate cloud like patterns from value noise. Could be used to create map like structures for games and such.
  • Unity Noise 2018. Similar to the above tool, but using Unity and the built in noise function (perlin)

Games

  • Platformer 2018. Simple proof of concept platforming engine made in Unity 2018. Graphics available here.
  • Mini Circle Wars (2013) My entry to the Ludum Dare 26. It's a tower defense game made with LibGDX.
  • Atoms (2000) A board game made back in high school inspired by an old public domain board game of the same name for the Atari ST. Programmed in C using libsdl with graphics made in Paint Shop Pro 4. Still runs on Windows 10.
