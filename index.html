Web experiments

  • Group Generator. Generate random groups of stuff from a custom list.
  • Java Source Printer. My attempt at making a web based Java code formatter with syntax highlighting using "highlight.js". Needs a rewrite.
  • Value Noise Generator. An old tool I made to generate cloud like patterns from value noise. The idea was to expand it to maps for games, but I never got that far.

Games

  • Mini Circle Wars (2013) My entry to the Ludum Dare 26. It's a tower defense game made with LibGDX.
  • Valhalla (2013) A prototype game for a platforming engine I made a while back, also utilizing LibGDX. Abondoned.
  • Atoms (2000) A board game made back in high school inspired by an old public domain board game of the same name for the Atari ST. Programmed in C using libsdl with graphics made in Paint Shop Pro 4. And it still runs flawlessly on Windows 10.
